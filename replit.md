# JDSA Students Bank

A Next.js application for managing student accounts, transactions, and academic records.

## Features
- Student management
- Transaction tracking (Deposits/Withdrawals)
- Bulk transaction editing
- Real-time updates with Pusher
- Academic year-based records
- QR code integration for student profiles
- Exportable reports (PDF/Excel)

## Tech Stack
- Next.js 15
- MongoDB with Mongoose
- Tailwind CSS
- Pusher for real-time updates
- Lucide React for icons
- Radix UI for components
